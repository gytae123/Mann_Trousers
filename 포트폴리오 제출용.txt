
- 타입(너비): 데스크탑 / 786px(태블릿) / 480px


- 메인페이지: 
 1. 초기 화면 화이트 페이드 인 효과(전 페이지 적용)
 2. 스와이퍼 사용 -> 메인화면 로테이션
 3. 로테이션 시 메시지 페이드인-업
 4. 왼쪽 상단 nav 호버시 한글 변환
 5. 오른쪽 상단 동그란 로고(가이드 버튼)은
    메인페이지로 이동시킴
 6. 스크롤 내리면 푸터
    (푸터 'follow us'에 각 외부 링크 총 2개, 전 페이지 적용)


- 2번 페이지(About):
 1. 가운데 로고는 페이지 스크롤에 따른 이벤트
 2. 스크롤링에 따른 배경 이미지 드러나는 이벤트


- 3번 페이지(Bespoke Service):
 1. 왼쪽 상단 nav 호버시 한글 변환
 2. 스크롤링에 따른 6개의 섹션 이벤트(텍스트 및 이미지)
 3. 페이지 맨 하단 '예약하기' 버튼 누르면 실제 외부링크 연결


- 4번 페이지(Archives):
 1. 왼쪽 상단 nav 호버시 효과 적용
 2. 슬라이드로 이미지 무한 로테이션
 3. 이미지 클릭시 각 알맞는 외부 링크로 이동


- 5번 페이지(Class)
 1. 왼쪽 상단 nav 호버시 효과 적용
 2. 페이지 중단에 아코디언 적용
 3. 하단에 이미지 변환 효과 적용된 박스 (7초마다 변환)

- 6번 페이지(Contact Us)
 1. 왼쪽 상단 nav 호버시 효과 적용
 2. 데이터 입력박스(이름, 이메일, 적을 내용) 적용,
    올바르지 않은 양식 입력 시 오류 메시지 호출
    (반대 시 '성공적으로 접수되었습니다.')
